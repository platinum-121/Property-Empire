# Property Empire Simulator

A property management simulation built with Python 3.11+ and PyQt6.

Core loop:
- Review property deals
- Buy or finance properties
- Collect rent
- Pay maintenance, loan interest, and taxes
- Expand into more cities
- Compete against property portfolio rivals

Current status:
Playable property-focused foundation with saves, city data, property purchases, rent simulation, loans, financial statements, news, and NPC competitors.
